# Headless WordPress & Next.js Research and Development Project

This repository is a research and development project focused on building high-performance headless WordPress websites using Next.js.

The project is being used to explore and test:

- Semantic HTML architecture
- Gutenberg block rendering in Next.js
- WPGraphQL data flow and content modelling
- Structured data and JSON-LD implementation
- Core Web Vitals and Lighthouse optimisation
- Accessibility and semantic document structure
- Dynamic component rendering patterns
- Headless CMS architecture
- Local business and entity-oriented SEO concepts
- Scalable frontend component systems
- Performance-conscious third-party embed handling
- Modern React and Next.js development workflows

## Technology Stack

- WordPress (Headless CMS)
- WPGraphQL
- Next.js
- React
- Tailwind CSS
- ACF Pro
- Vercel (planned deployment)
- GitHub version control

## Current Areas of Development

- Gutenberg block rendering system
- Dynamic BlockRenderer architecture
- Reusable layout and content components
- Semantic list rendering (`ul` / `li`)
- Structured data implementation
- Image optimisation and performance tuning
- Entity-focused page architecture
- Accessibility improvements
- Site performance and rendering analysis

## Project Goals

The primary goal of this project is to investigate how modern headless WordPress architectures can be used to create:

- Fast, scalable frontend experiences
- Semantically rich HTML structures
- SEO-conscious rendering patterns
- Structured and machine-readable content
- Maintainable component-based systems
- Flexible content modelling workflows
